The Question
"What if Tandy had made better
software decisions in 1982?"

The hardware was never the problem. The cassette interface, the 32-column editor, the assembler that couldn't handle a real project — those were. And they were all software.

The Hardware

The Motorola 6809

  • Two 16-bit index registers — X and Y
  • Two stack pointers — S and U
  • PC-relative addressing — position-independent code is natural
  • Clean, orthogonal ISA — influenced by the PDP-11
  • Writing 6809 assembly is actually pleasant
The 6809 was running multitasking operating systems in the early 1980s. The hardware had headroom. The tooling didn't keep up.

Why Forth?

The Perfect Fit

Forth is defined by two stacks:

  • A data stack for working values
  • A return stack for tracking word calls

The 6809 has two hardware stack pointers: S and U.

This is not a coincidence. The 6809 was practically designed to run Forth.

The entire kernel fits in ~1KB of 6809 assembly. Most Forth implementations for the CoCo need a disk and an OS. This one needs neither.

The Architecture

Indirect Threaded Code (ITC)

; The entire inner interpreter: LDY ,X++ ; fetch next CFA, advance IP JMP [,Y] ; jump through CFA to machine code
  • Kernel — ~1KB of 6809 assembly, 25 primitives
  • CFA dispatch table — maps tokens to machine code
  • Compiled thread — a sequence of 2-byte addresses
  • No dictionary on the CoCo
  • No REPL, no OS, no overhead
Register assignments: X = IP instruction pointer U = DSP data stack pointer S = RSP return stack pointer
Memory map: $1000 kernel + CFA table $2000 your compiled Forth $7E00 data stack (grows ↓) $8000 return stack (grows ↓)

Where This Goes

Roadmap

  • Serial loader — bit-banged via PIA, load bytecode over RS-232
  • ROM cartridge image — kernel burned to flash, plug-and-play
  • SD card — store and load programs via CoCoSDC
  • RP2350 co-processor — one core on the 6809 bus, one managing the outside world
  • CoCo 1 / 2 / 3 — same bytecode runs everywhere; hardware differences are the kernel's problem
