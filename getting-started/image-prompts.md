# Getting Started — Illustration Image Prompts

Each entry shows the target filename, the illustration type (for sizing reference),
and the prompt text from the HTML placeholder.

To add a finished image, replace the placeholder div with the pattern from Chapter 9:
```html
<div class="illustration illustration-TYPE" style="padding:0;border:none;background:none;">
  <img src="images/FILENAME" alt="DESCRIPTION" style="width:100%;height:auto;display:block;">
</div>
```
For openers use `height:100%; object-fit:cover` instead of `height:auto`.
