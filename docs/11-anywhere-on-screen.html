ANYWHERE ON SCREEN 1

The Cursor Problem

Every word you've written so far prints at the cursor and advances it. That's fine for running text, but it makes layout impossible — you can't put a label at column 8 and a value at column 20 on the same row without printing exactly the right number of spaces in between.

The fix is to set the cursor directly. AT takes a column and row and jumps the cursor there instantly, no spaces needed.

AT — Jump the Cursor

AT  ( col row -- )

Set the cursor to column col, row row. The next EMIT or CR writes there.

10  3 AT  CHAR H EMIT   \ 'H' at column 10, row 3
 0  0 AT  CHAR * EMIT   \ '*' at top-left corner
31 15 AT  CHAR . EMIT   \ '.' at bottom-right corner

The Screen Is a Grid

The CoCo's text screen is 32 columns wide and 16 rows tall. Columns run 0–31 left to right; rows run 0–15 top to bottom. AT computes the offset as row × 32 + col and stores it in the cursor register — that's all it does.

Position AT call Offset
top-left 0 0 AT0
col 10, row 310 3 AT3×32+10 = 106
bottom-right31 15 AT15×32+31 = 511

After AT, the cursor is set. Subsequent EMIT calls advance it normally — so you only need one AT to start a run of text; you don't need AT for every character.

ANYWHERE ON SCREEN 2

Drawing Lines with AT + DO

AT positions the cursor once; a DO loop emits the rest. This gives you a clean word for drawing horizontal bars: 

: HBAR  ( len col row -- )
  AT  0 DO CHAR - EMIT LOOP ;

AT pops col and row, leaving len on the stack. The loop emits len dashes. The cursor advances automatically after each EMIT, so the dashes appear in a row without a second AT. 

20  4  2 HBAR   \ 20 dashes starting at col 4, row 2
20  4  4 HBAR   \ same at row 4 — a second bar below

Combining AT and Text

Once you have AT, you can build any fixed layout: labels, borders, status displays. Call AT to jump to the right spot, then emit your text normally. Here is a word that draws a labelled divider: 

: DIVIDER  ( row -- )
  22  SWAP  HBAR         \ 22 dashes from col 4
   4  OVER AT            \ back to col 4, same row
  CHAR [ EMIT
  22  OVER 1 + AT        \ col 22, same row
  CHAR ] EMIT
  DROP ;

The rule: AT for position, then EMIT/CR for content. One AT starts a run; it does not need to be repeated for every character.

: MAIN
  \ Title centred on row 3
  10  3 AT
  CHAR C EMIT CHAR O EMIT CHAR L EMIT CHAR O EMIT CHAR R EMIT
  32 EMIT
  CHAR F EMIT CHAR O EMIT CHAR R EMIT CHAR T EMIT CHAR H EMIT

  \ Underline the title
  11 10  4 HBAR

  \ Tagline on row 6
  10  6 AT
  CHAR O EMIT CHAR N EMIT 32 EMIT
  CHAR Y EMIT CHAR O EMIT CHAR U EMIT CHAR R EMIT 32 EMIT
  CHAR C EMIT CHAR O EMIT CHAR C EMIT CHAR O EMIT

  \ Park cursor at bottom-left
  0 15 AT ;
